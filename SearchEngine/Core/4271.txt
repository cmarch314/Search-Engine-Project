files. Every account\r\n          start off with basic statup files. These files may have been removed\r\n          mistakenly if the user was not careful. </p>\r\n        <p>\r\n          The following files MUST be in your home directory (H-drive):</p>\r\n        <ul>\r\n          <li> .login</li>\r\n          <li> .logout</li>\r\n          <li> Depending on which shell you're using, you will also have\r\non of the following file:<br>\r\n            <ul>\r\n              <li>.cshrc (<i>If you've never changed your shell, this is the default one setup for all users.</i>)</li>\r\n              <li>.bashrc</li>\r\n              <li>.tcshrc</li>\r\n              <li>.zshrc</li>\r\n            </ul>\r\n          </li>\r\n        </ul>\r\n        <p>If one or all of the above files is missing, you will have problems\r\n          logging onto the ICS hosts.</p>\r\n        <p>You can add them by following these steps:</p>\r\n        <ol>\r\n          <li> SSH to openlab.ics.uci.edu from a Windows machine<br>\r\n            <ul>\r\n\t\t\t  <li>If you can NOT SSH to openlab, have the lab assistant reset your\r\n            password and try again.</li>\r\n              <li>If it still does not work, please ask the lab assistant to file\r\n                an online report form.</li>\r\n\t\t\t</ul>\r\n\t\t  </li>\r\n\t\t  <br>\r\n          <li>Type in the following command at the prompt:<br>\r\n            <ul>\r\n\t\t\t  <li><font face=\"Courier New, Courier, mono\">%cd /opt/local/etc/skel/</font></li>\r\n\t\t\t</ul>\r\n\t\t  </li>\r\n\t\t  <br>\r\n          <li>Copy all missing files ending with &quot;@&quot; to /home/&quot;login_name&quot;/.&quot;filename&quot;<br>\r\n            Remember to put the dot before the destination file.<br>\r\n            <ul>\r\n\t\t\t  <li>For example, if you are missing the .cshrc file you would type the\r\n            following at the prompt:</li>\r\n\r\n              <ul>\r\n                <li><font face=\"Courier New, Courier, mono\">%cp cshrc /home/&quot;login_name&quot;/.cshrc</font></li>\r\n              </ul>\r\n              <li>You may also want to look at the other <font face=\"Courier New, Courier, mono\">example.*</font> files as some have more options than other that you may like to use.</li>\r\n            </ul>\r\n          </li>\r\n\t\t  <br>\r\n          <li>After copying all the files you are missing, return to your home\r\n            directory by typing the following command at the prompt:<br>\r\n            <ul>\r\n              <li><font face=\"Courier New, Courier, mono\">%cd</font></li>\r\n            </ul>\r\n\t\t  </li>\r\n\t\t  <br>\r\n          <li>Finally, you need to <font face=\"Courier New, Courier, mono\">chmod</font>\r\n            all the files you copied. This basically means changing the permission\r\n            on the file so that it is only changeable by you. Type the following\r\n            command for each of the files you copied:\r\n            <ul>\r\n\t\t\t  <li><font face=\"Courier New, Courier, mono\">%chmod 744 .&quot;filename&quot;</font></li>\r\n\t\t\t  <li>For example, for the cshrc file, it would be:<ul>\r\n                  <li><font face=\"Courier New, Courier, mono\">%chmod 744 .cshrc</font></li>\r\n                </ul></li>\r\n\t\t\t</ul>\r\n          </li>\r\n\t\t  <br>\r\n          <li>You're DONE! </li>\r\n        </ol>\r\n        <p><em>If you have any questions regarding the above procedure,
