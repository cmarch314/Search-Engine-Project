Abstract \n<!----------------------------------------------------------------------->\n<H2>Abstract</H2>\n<p>\nData giving characteristics of each ORF (potential gene) in the\nE. coli genome. Sequence, homology (similarity to other genes) and\nstructural information, and function (if known) are provided.\n</p>\n\n<!------------------------------------------------------------------------\n<!  Sources \n<!----------------------------------------------------------------------->\n<H2> Sources</H2>\n<H4> Original Owner and Donor</H4>\n<PRE>\n  <a href=\"http://users.aber.ac.uk/rdk\">Ross D. King</a>\n  Department of Computer Science, \n  University of Wales Aberystwyth, \n  SY23 3DB, Wales\n  <a href=\"mailto:rdk@aber.ac.uk\">rdk@aber.ac.uk</a>\n</PRE>\n<B>Date Donated: </B> July 14, 2001\n\n<!------------------------------------------------------------------------\n<!  Data Characteristics\n<!----------------------------------------------------------------------->\n<H2> Data Characteristics</H2>\n<P>\nThe data was collected from several sources, including <a href=\"http://genprotec.mbl.edu/start\">GenProtEC</a> and <a href=\"http://www.expasy.ch/sprot/sprot-top.html\">SWISSPROT</a>. Structure prediction was made by <a href=\"http://www.aber.ac.uk/~phiwww/prof/index.html\">PROF</a>. Homology search was provided by <a href=\"http://www.ncbi.nlm.nih.gov/BLAST/\">PSI-BLAST</a>.\n\n<p>\nThe data is in Datalog format. Missing values are not explicit, but some \ngenes have more relationships than others. \n\n\n<H4>Dependencies</H4>\n\n<p>\nE. coli genes (ORFs) are related to each other by the predicate \necoli_to_ecoli(EcoliNumber,E-value,Psi-blast_iteration). \nThey are related to other (SWISSPROT) proteins by the predicate\ne_val(AccNo,E-value).\nAll the data for a single gene (ORF) is enclosed between delimiters of \nthe form:\n<pre>\n   begin(model(EcoliNumber)).\n   end(model(EcoliNumber)). \n</pre>\n\n\n\n<!------------------------------------------------------------------------\n<!  Other Relevant Information \n<!----------------------------------------------------------------------->\n<H2> Other Relevant Information</H2>\n\nThe gene functional classes are in a hierarchy. See \n<a href=\"http://genprotec.mbl.edu/start\">http://genprotec.mbl.edu/start</a>\n(note: the classes may have changed \nsince original data collection).\n\n\n<!------------------------------------------------------------------------\n<!  Data Format  \n<!----------------------------------------------------------------------->\n<H2>Data Format</H2>\n\nThere are two datalog files: ecoli_data.pl and ecoli_functions.pl\n\n<h4>ecoli_functions.pl</h4>\n<p>\nLists classes and ORF functions. Lines are of the following form:\n<pre>\n   class(5,1,1,'Colicin-related functions').\n   class(5,1,'Laterally acquirred elements').\n   class(5,'Extrachromosomal'). \n</pre>\n\n<p>\nArguments are up to 3 numbers (describing class at up to 3\ndifferent levels), followed by a string class description. For\nexample:\n<pre>\n   function(ecoli210,7,0,0,'b0217','putative aminopeptidase').\n</pre>\n\n<p>\nArguments are ORF number, exactly 3 class numbers, gene name (or\nblattner number if no gene name), ORF description.\n\n<h4>ecoli_data.pl</h4>\n<p>\nData for each ORF (gene) is delimited by \n<pre>\n   begin(model(ecoliX)).\n   end(model(ecoliX)).\n</pre>\nwhere X is the ORF number. Other predicates are as follows (examples):\n\n<PRE>\n   ecoli_orf(ecoliX).    % X is ORF number\n   ecoli_mol_wt(176624.1).  % float\n   ecoli_theo_pI(5.81).     %float\n   ecoli_atomic_comp(c,7940).   % {c,h,n,o,s} , int\n   ecoli_aliphatic_index(69.57). % float\n   ecoli_hydro(-0.549).          % float\n   sec_struc(1,c,2).           % int (start), {a,b,c}, int (length)\n   sec_struc_coil(1,2).        % int (start), int (length)\n   sec_struc_beta(1,5).        % int (start), int (length)\n   sec_struc_alpha(1,7).       % int (start), int (length)\n   sequence_length(255).       % int\n   amino_acid_ratio(a,8.9).    % amino_acid_char, float\n   amino_acids(ecoli3013,a,70). % ORF_num, amino_acid_char, int\n   amino_acid_pair_ratio(a,a,9.0). % amino_acid_char, amino_acid_char, float\n   amino_acid_pairs(a,a,7).    % amino_acid_char, amino_acid_char, int\n   ecoli_to_ecoli(1170,1.0e-105,5).  % ORF_num, double (e-value), int (iteration) \n   e_val(o42893,2.0e-99).  % accession_number, double (e-value)\n   psi_iter(o42893,5).     % accession_number, int (iteration)\n   species(p52494,'candida_albicans__yeast_').  % accession_number, string\n   mol_wt(p52494,104022). % accession_number, int \n   classification(p52494,candida).  % accession_number, name\n   keyword(p25195,'plasmid').   % accession_number, string\n</PRE>\n\n\n\n<!------------------------------------------------------------------------\n<!
