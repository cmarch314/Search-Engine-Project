name=\"internal_hyperlinks\" id=\"internal_hyperlinks\">Internal hyperlinks</a></h2>\n<div class=\"level2\">\n\n<p>\nYou should always refer to figures, sections, theorems, etc. by label instead of by number. These labels are created by code like\n</p>\n<pre class=\"code latex\"><span class=\"re8\">\\begin</span><span class=\"sy0\">{</span><span class=\"re9\"><span class=\"re7\">theorem</span></span><span class=\"sy0\">}</span>\n<span class=\"sy0\">\\</span><a href=\"http://www.golatex.de/wiki/index.php?title=%5Clabel\"><span class=\"kw1\">label</span></a><span class=\"sy0\">{</span><span class=\"re9\">thm:my-theorem-name</span><span class=\"sy0\">}</span>\nThe statement of the theorem goes here.\n<span class=\"re8\">\\end</span><span class=\"sy0\">{</span><span class=\"re9\"><span class=\"re7\">theorem</span></span><span class=\"sy0\">}</span></pre>\n\n<p>\nor\n</p>\n<pre class=\"code latex\"><span class=\"sy0\">\\</span><a href=\"http://www.golatex.de/wiki/index.php?title=%5Csection\"><span class=\"kw1\">section</span></a><span class=\"sy0\">{</span><span class=\"re9\">Section title</span><span class=\"sy0\">}</span>\n<span class=\"sy0\">\\</span><a href=\"http://www.golatex.de/wiki/index.php?title=%5Clabel\"><span class=\"kw1\">label</span></a><span class=\"sy0\">{</span><span class=\"re9\">sec:my-section-name</span><span class=\"sy0\">}</span></pre>\n\n<p>\netc. This helps keep the numbers matching up correctly when things get renumbered, but it can also be used to make internal hyperlinks so that clicking on the name of a thing will take you to the thing. To do this, include this in your preamble:\n</p>\n<pre class=\"code latex\"><span class=\"sy0\">\\</span><a href=\"http://www.golatex.de/wiki/index.php?title=%5Cusepackage\"><span class=\"kw1\">usepackage</span></a><span class=\"sy0\">{</span><span class=\"re9\">hyperref</span><span class=\"sy0\">}</span></pre>\n\n<p>\nThen, instead of\n</p>\n<pre class=\"code latex\">Theorem~<span class=\"sy0\">\\</span><a href=\"http://www.golatex.de/wiki/index.php?title=%5Cref\"><span class=\"kw1\">ref</span></a><span class=\"sy0\">{</span><span class=\"re9\">thm:name-of-theorem</span><span class=\"sy0\">}</span></pre>\n\n<p>\n(which is how you would refer to the label without hyperref), use\n</p>\n<pre class=\"code latex\"><span class=\"re12\">\\autoref</span><span class=\"sy0\">{</span><span class=\"re9\">thm:name-of-theorem</span><span class=\"sy0\">}</span></pre>\n\n<p>\nAs well as making it easier to change a lemma to a theorem or \u201cFigure\u201d to \u201cFig.\u201d (etc), this produces better hyperlinks: the link covers the whole \u201cTheorem 3\u201d text not just the number. In order to get this to work properly for labels that the package doesn&#039;t already know about, you may need additional preamble code like\n</p>\n<pre class=\"code latex\"><span class=\"sy0\">\\</span><a href=\"http://www.golatex.de/wiki/index.php?title=%5Crenewcommand\"><span class=\"kw1\">renewcommand</span></a><span class=\"sy0\">{</span><span class=\"re9\"><span class=\"re12\">\\sectionautorefname</span></span><span class=\"sy0\">}{</span><span class=\"re9\">Section</span><span class=\"sy0\">}</span></pre>\n\n<p>\nLabels also work for the items of an enumeration environment, but you may want to use \\ref instead of \\autoref for these, since the type of thing you want to call it (Case, Property, etc) will probably vary from one enumeration to another.\n</p>\n\n</div>\n<!-- SECTION \"Internal hyperlinks\" [629-2091] -->\n<h2><a name=\"number_lines_for_submissions\" id=\"number_lines_for_submissions\">Number lines for submissions</a></h2>\n<div class=\"level2\">\n\n<p>\nIt is convenient for paper reviewers to have line numbers to refer to when discussing proposed changes to your paper.\nThe easy way to do this is to put\n</p>\n<pre class=\"code latex\"><span class=\"sy0\">\\</span><a href=\"http://www.golatex.de/wiki/index.php?title=%5Cusepackage\"><span class=\"kw1\">usepackage</span></a><span class=\"sy0\">{</span><span class=\"re9\">lineno</span><span class=\"sy0\">}</span></pre>\n\n<p>\nin the preamble, and to put\n</p>\n<pre class=\"code latex\"><span class=\"re12\">\\linenumbers</span></pre>\n\n<p>\n in the document itself, before the abstract.\n</p>\n\n</div>\n<!-- SECTION \"Number lines for submissions\" [2092-2432] -->\n<h2><a name=\"less_than_and_greater_than_signs_in_text_mode\" id=\"less_than_and_greater_than_signs_in_text_mode\">Less than and greater than signs in text mode</a></h2>\n<div class=\"level2\">\n\n<p>\nWith the default settings, LaTeX turns &lt; and &gt; in text-mode input into \u00a1 and \u00bf.
