using the RSS (Really Simple Syndication) XML format.\nUsing the RSS subscription feature in Trac, you can easily monitor progress of the project, a set of issues or even changes to a single file.\n</p>\n<p>\nTrac supports RSS feeds in:\n</p>\n<ul><li><a class=\"wiki\" href=\"/LUCICodeRepository/nomaticIM/wiki/TracTimeline\">TracTimeline</a> \u2014  Use the RSS feed to <strong>subscribe to project events</strong>.<br />Monitor overall project progress in your favorite RSS reader.\n</li><li><a class=\"wiki\" href=\"/LUCICodeRepository/nomaticIM/wiki/TracTickets\">TracTickets</a>, <a class=\"wiki\" href=\"/LUCICodeRepository/nomaticIM/wiki/TracReports\">TracReports</a>, and <a class=\"wiki\" href=\"/LUCICodeRepository/nomaticIM/wiki/TracQuery\">TracQuery</a> \u2014 Allows syndication of report and ticket query results.<br />Be notified about important and relevant issue tickets.\n</li><li><a class=\"wiki\" href=\"/LUCICodeRepository/nomaticIM/wiki/TracBrowser\">TracBrowser</a> and <a class=\"wiki\" href=\"/LUCICodeRepository/nomaticIM/wiki/TracRevisionLog\">TracRevisionLog</a> \u2014 Syndication of file changes.<br />Stay up to date with changes to a specific file or directory.\n</li></ul><h2 id=\"HowtoaccessRSSdata\">How to access RSS data</h2>\n<p>\nAnywhere in Trac where RSS is available, you should find a small orange <strong>XML</strong> icon, typically placed at the bottom of the page. Clicking the icon will access the RSS feed for that specific resource.\n</p>\n<p>\n<strong>Note:</strong> Different modules provide different data in their RSS feeds. Usually, the syndicated information corresponds to the current view. For example, if you click the RSS link on a report page, the feed will be based on that report. It might be explained by thinking of the RSS feeds as an <em>alternate view of the data currently displayed</em>.\n</p>\n<h2 id=\"Links\">Links</h2>\n<ul><li><em>Specifications:</em>\n<ul><li><a
