Both of these functions require that\nOEFindRingAtomsAndBonds has previously been called on the molecule.\nBoth of these functions take the query ring size as an argument, which\nshould be greater than or equal to three.  The definition of ring or\ncycle is not based upon the SSSR, and these return true if there is\na bonded path of `size' unique atoms where each atom is bonded to the\nnext and the last is bonded to the first.\n\n<P>\nIt is often the case that atoms may be in different sized cycles at\nthe same time.
