The interfaces do not inherit from <code>java.lang.Object</code>.<UL>\n<LI>When viewing the Overview page, clicking on \"Tree\" displays the hierarchy for all packages.<LI>When viewing a particular package, class or interface page, clicking \"Tree\" displays the hierarchy for only that package.</UL>\n</BLOCKQUOTE>\n<H3>\nDeprecated API</H3>\n<BLOCKQUOTE>\nThe <A HREF=\"deprecated-list.html\">Deprecated API</A> page lists all of the API that have been deprecated.
