class=\"wikipage searchable\">\n        \n          <div id=\"wikipage\" class=\"trac-content\"><h1 id=\"UsingSearch\">Using Search</h1>\n<p>\nTrac has a built-in search engine to allow finding occurrences of keywords and substrings in wiki pages, tickets and changeset properties (author, revision and log message).\n</p>\n<p>\nUsing the Trac search facility is straightforward and its interface should be familiar to most users.\n</p>\n<p>\nApart from the <a class=\"search\" href=\"/LUCICodeRepository/nomaticIM/search\">Search module</a>, you will also find a small search field above the navigation bar at all time. It provides convenient access to the search module from all pages.\n</p>\n<h2 id=\"Quickjumpsearches\">\"Quickjump\" searches</h2>\n<p>\nFor quick access to various project resources, the quick-search field at the top of every page can be used to enter a <a class=\"wiki\" href=\"/LUCICodeRepository/nomaticIM/wiki/TracLinks\">wiki link</a>, which will take you directly to the resource identified by that link.\n</p>\n<p>\nFor example:\n</p>\n<ul><li>[42] -- Opens change set 42\n</li><li>#42 -- Opens ticket number 42\n</li><li>{1} -- Opens report 1\n</li><li>/trunk -- Opens the browser for the <tt>trunk</tt> directory\n</li></ul><h2 id=\"Advanced\">Advanced</h2>\n<h3 id=\"DisablingQuickjumps\">Disabling Quickjumps</h3>\n<p>\nTo disable the quickjump feature for a search keyword - for example when searching for occurences of the literal word TracGuide - begin the query with an exclamation mark (!).\n</p>\n<h3 id=\"SearchLinks\">Search Links</h3>\n<p>\nFrom the Wiki, it is possible to link to a specific search, using\n<tt>search:</tt> links:\n</p>\n<ul><li><tt>search:?q=crash</tt> will search for the string \"crash\" \n</li><li><tt>search:?q=trac+link&amp;wiki=on</tt> will search for \"trac\" and \"link\" \nin wiki pages only\n</li></ul><hr />\n<p>\nSee also: <a class=\"wiki\" href=\"/LUCICodeRepository/nomaticIM/wiki/TracGuide\">TracGuide</a>, <a class=\"wiki\"
