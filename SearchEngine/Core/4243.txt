data-intensive computing,\ndata integration and sharing, data warehouses, data cleansing,\nand Web information management.  The following are several database\nprojects I am working on. <BR>\n\n<h3>Current Projects</h3>\n<ul>\n<li> <a href=\"http://asterix.ics.uci.edu/\">ASTERIX</a>: Mike Carey and I are starting this project on \nActive, Scalable, Transactional Enterprise Repository for Information in XML. It's a new effort to develop a scalable semi-structured information management system, based on XML and XQuery technologies, targeting very large shared-nothing compute clusters.\n\n<li><a href=\"http://flamingo.ics.uci.edu/\">FLAMINGO</a>: A project on data cleaning. \n\n<li><a href=\"http://ipubmed.ics.uci.edu/\">IPUBMED</a>: Efficient instant\n  search on large amounts of data.
