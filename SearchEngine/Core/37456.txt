This section describes molecular graphics software\n        which can display potentials and other data output from APBS.\n      </P\n><DIV\nCLASS=\"sect3\"\n><H3\nCLASS=\"sect3\"\n><A\nNAME=\"AEN2471\"\n></A\n>PMV</H3\n><P\n>PMV \n          (<A\nHREF=\"http://www.scripps.edu/~sanner/python\"\nTARGET=\"_top\"\n>http://www.scripps.edu/~sanner/python</A\n>)\n          is a Python-based molecular visualization package which provides an\n          interface to APBS.  It not only permits isocontour- and surface-based\n          visualization of APBS results and it also integrates setup and\n          executation of APBS calculations.  The PMV/APBS interface is under\n          active development and future versions will offer even more setup,\n          visualization, and analysis functionality.\n        </P\n><P\n>The APBS interface is distributed with recent beta versions of\n          PMV, avalaible from \n          <A\nHREF=\"http://www.scripps.edu/~sanner/python\"\nTARGET=\"_top\"\n>http://www.scripps.edu/~sanner/python</A\n>.\n        </P\n></DIV\n><DIV\nCLASS=\"sect3\"\n><H3\nCLASS=\"sect3\"\n><A\nNAME=\"AEN2477\"\n></A\n>VMD</H3\n><P\n>VMD (<A\nHREF=\"http://www.ks.uiuc.edu/Research/vmd/\"\nTARGET=\"_top\"\n>http://www.ks.uiuc.edu/Research/vmd/</A\n>)\n          is a molecular visualization and animation package which provides an\n          interface to APBS.  It permits isocontour-based visualization of APBS\n          results and provides a Tcl plugin to setup APBS calculations for\n          visualization.\n        </P\n><P\n>The APBS interface is provided as a plugin to VMD and is\n          available from <A\nHREF=\"http://www.ks.uiuc.edu/Research/vmd/plugins/apbsrun/\"\nTARGET=\"_top\"\n>http://www.ks.uiuc.edu/Research/vmd/plugins/apbsrun/</A\n>.\n        </P\n><P\n>&#13;          Additionally, APBS includes simple visualization-only Tcl\n          scripts for VMD in the\n          <TT\nCLASS=\"filename\"\n>tools/visualization/vmd</TT\n> (see the discussion of\n          <A\nHREF=\"x452.html#viz-tools\"\n>Data visualization tools</A\n> in this\n          manual). \n        </P\n></DIV\n><DIV\nCLASS=\"sect3\"\n><H3\nCLASS=\"sect3\"\n><A\nNAME=\"AEN2486\"\n></A\n>PyMOL</H3\n><P\n>PyMOL (<A\nHREF=\"http://pymol.org/\"\nTARGET=\"_top\"\n>http://pymol.org/</A\n>)\n          Python-based a molecular graphics program. A program is provided with APBS (see\n          <TT\nCLASS=\"filename\"\n>tools/mesh</TT\n> and the\n          <A\nHREF=\"x452.html#conversion-sect\"\n>Data conversion tools</A\n> in this\n          manual) which converts OpenDX data to MOLMOL format.\n        </P\n></DIV\n><DIV\nCLASS=\"sect3\"\n><H3\nCLASS=\"sect3\"\n><A\nNAME=\"AEN2505\"\n></A\n>OpenDX</H3\n><P\n>OpenDX (<A\nHREF=\"http://www.opendx.org\"\nTARGET=\"_top\"\n>http://www.opendx.org</A\n>)\n          is a general data visualization package which can read APBS output\n          using the scripts provided in\n          <TT\nCLASS=\"filename\"\n>tools/visualization/opendx</TT\n> (see the discussion\n          of <A\nHREF=\"x452.html#viz-tools\"\n>Data visualization tools</A\n> in this\n          manual).
