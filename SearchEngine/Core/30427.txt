tags file for use with the UNIX\r\neditors ex and vi, similar to <i> ctags(1)</i>.\r\nThe library is loaded using\r\n<blockquote> <PRE>:- lib(ptags).\r\n</PRE>\r\n</blockquote>\r\nand the predicates <PRE>:- ptags(+File)\r\n</PRE>\r\n</blockquote>\r\nor\r\n<blockquote> <PRE>:- ptags(+File, +TagsFile)\r\n</PRE>\r\n</blockquote>\r\n<i> +TagsFile</i> is the name\r\nof the tags file. If <i> +TagsFile</i>\r\nis omitted, it defaults to <tt> tags</tt>.\r\n<P>\r\nThe tags file created by the <b> ptags/1, 2</b> predicates\r\n<A NAME=6973>&#160;</A>\r\ncan be used as a tags file for <i> vi</i> or <i> ex</i>.\r\nA procedures specified as <i> Name/Arity</i>\r\ncan be found using the command\r\n<blockquote> <PRE>:ta Name\r\n</PRE>\r\n</blockquote>\r\nIf there are several procedures with the same name and different arity,\r\nthe above command will find only one of them.\r\nIn this case\r\nthe command\r\n<blockquote> <PRE>:ta Name/Arity\r\n</PRE>\r\n</blockquote>\r\nshould be used.\r\nIf the clauses for the procedure are not consecutive or if the procedure\r\noccurs in more than one file, only one occurrence will be put into\r\nthe tags file.\r\nWhich one it will be depends on the file name and on the contents\r\nof the line that is being sought by the :ta command.\r\n<P>\r\n<BR> <HR><A NAME=tex2html5827 HREF=\"node298.html\"><IMG ALIGN=BOTTOM ALT=\"next\" SRC=\"http://www.ecrc.de/eclipse/new_icons//next_motif.gif\"></A>   <A NAME=tex2html5825 HREF=\"node268.html\"><IMG ALIGN=BOTTOM ALT=\"up\" SRC=\"http://www.ecrc.de/eclipse/new_icons//up_motif.gif\"></A>   <A NAME=tex2html5819 HREF=\"node296.html\"><IMG ALIGN=BOTTOM ALT=\"previous\" SRC=\"http://www.ecrc.de/eclipse/new_icons//previous_motif.gif\"></A>      <A NAME=tex2html5829 HREF=\"node321.html\"><IMG ALIGN=BOTTOM ALT=\"index\" SRC=\"http://www.ecrc.de/eclipse/new_icons//index_motif.gif\"></A>   <BR>\r\n<B> Next:</B> <A NAME=tex2html5828 HREF=\"node298.html\"> Quintus Prolog Compatibility </A>\r\n<B>Up:</B> <A NAME=tex2html5826 HREF=\"node268.html\"> Libraries</A>\r\n<B> Previous:</B> <A NAME=tex2html5820 HREF=\"node296.html\"> Parallel Utilities</A>\r\n<BR> <HR> <P>\r\n<BR> <HR>\r\n<P><ADDRESS>\r\n<I>Micha Meier <BR>\r\nMon Mar  4 12:11:45 MET 1996</I>\r\n</ADDRESS>\r\n</BODY>\r\n", "id": 30427.0}
