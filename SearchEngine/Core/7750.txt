The AsterixDB BDMS is the result of about 3.5 years\r\n            of R&D involving researchers at UC Irvine, UC Riverside, and UC San\r\n            Diego. The project has been combining ideas\r\n            from three distinct areas\u2014semi-structured data, parallel databases,\r\n            and data-intensive computing (a.k.a. today\u2019s Big Data platforms)\u2014in\r\n            order to create a next-generation, open-source software platform\r\n            that scales by running on large, shared-nothing commodity computing\r\n            clusters.</p>\r\n\r\n            <p>The ASTERIX effort has been targeting a wide range of\r\n            semi-structured information, ranging from \u201cdata\u201d use cases\u2014where\r\n            information is well-typed and highly regular\u2014to \u201ccontent\u201d use\r\n            cases\u2014where data tends to be irregular, much of each datum may be\r\n            textual, and the ultimate schema for the various data types\r\n            involved may be hard to anticipate up front. The ASTERIX project\r\n            has been addressing technical issues including highly scalable data\r\n            storage and indexing, semi-structured query processing on very\r\n            large clusters, and merging time-tested parallel database\r\n            techniques with modern data-intensive computing techniques to\r\n            support performant yet declarative solutions to the problem of\r\n            storing and analyzing semi-structured information effectively.</p>\r\n            \r\n            <p>The first fruits of this labor have been captured in the\r\n            AsterixDB system that is now being released in preliminary or\r\n            \u201cBeta\u201d release form.
