The request for stereochemistry\nof a particular class can be made of an <code>OEBondBase</code> using the\nfunction <code>HasStereoSpecified(unsigned int type)</code>.  The type argument\nprovided to <code>HasStereoSpecified()</code> indicates the type of\nstereochemistry requested, and must be one of the constants listed in\nthe <code>OEBondStereo</code> namespace.\nThe most commonly requested stereochemistry value of bonds is whether\nthe configuration of two atoms around a non-rotatable bond is cis or\ntrans.  The following code sample demonstrates a loop over bonds which\ntests for bonds with associated stereochemistry, and retrieval of\nwhether the neighboring atoms are cis or trans relative to one\nanother.\n\n<P>\n<dl><dd><pre class=\"verbatim\">#include \"oechem.h\"\n#include &lt;iostream&gt;\n#include &lt;vector&gt;\n\nusing namespace OESystem;\nusing namespace OEChem;\nusing namespace std;\n\nint main()\n{\n  OEMol mol;\n  OEParseSmiles(mol, \"C/C=C/C\");\n\n  OEIter&lt;OEBondBase&gt; bond;\n  for (bond = mol.GetBonds();bond;++bond)\n  {\n    if (bond-&gt;HasStereoSpecified(OEBondStereo::CisTrans))\n    {\n      OEIter&lt;OEAtomBase&gt; atom1,atom2;\n      for (atom1 = bond-&gt;GetBgn()-&gt;GetAtoms();atom1;++atom1)\n        if (atom1 != bond-&gt;GetEnd())\n        {\n          for (atom2 = bond-&gt;GetEnd()-&gt;GetAtoms();atom2;++atom2)\n            if (atom2 != bond-&gt;GetBgn())\n            {\n              std::vector&lt;OEAtomBase*&gt; v;\n              v.push_back(atom1);\n              v.push_back(atom2);\n\n              unsigned int stereovalue =\n                bond-&gt;GetStereo(v,OEBondStereo::CisTrans);\n\n              cerr &lt;&#60; \"Atoms: \" &lt;&#60; atom1-&gt;GetIdx() &lt;&#60; \" and \";\n              cerr &lt;&#60; atom2-&gt;GetIdx() &lt;&#60; \" are \";\n\n              if (stereovalue == OEBondStereo::Cis)\n                cerr &lt;&#60; \"cis\" &lt;&#60; endl;\n              else if (stereovalue == OEBondStereo::Trans)\n                cerr &lt;&#60; \"trans\" &lt;&#60; endl;\n              else\n                cerr &lt;&#60; \"neither cis nor trans\" &lt;&#60; endl;\n            }\n        }\n    }\n  }\n  return 0;\n}\n\n</pre>\n<div class=\"verbatiminput-footer\">\n<a href=\"ch14-3.txt\" type=\"text/plain\">Download as text.</a>\n</div>\n</dd></dl>\n\n<P>\nStereochemistry values can be associated with a bond using the\n<code>SetStereo(const std::vector&lt;OEAtomBase*&gt; &amp;,unsigned int type, unsigned\nint value)</code> function.  The STL vector must contain atoms which are\nconnected to the atoms on either end of the bond on which\nstereochemistry is being set.
