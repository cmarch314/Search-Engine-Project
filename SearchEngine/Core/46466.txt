bit about multimedia information retrieval.  So there maybe a few posts related to this topic.</p>\n<p>I will start by saying that it has been nice to see that there is a nice data set that is collected and could be used for testing and evalyation purposes.  This data set is available from http://cophir.isti.cnr.it/  .  They claim to have collected 100 Million images from Flickr in this collection.  The following is the abstract of the white paper on this site &#8212; giving idea about the collection:</p>\n<blockquote><p>As the number of digital images is growing fast and Content-based<br />\nImage Retrieval (CBIR) is gaining in popularity, CBIR systems should<br />\nleap towards Web-scale datasets. In this paper, we report on our experience<br />\nin building a test collection of more than 50 million images, with<br />\nthe corresponding descriptive features, to be used in experimenting new<br />\ntechniques for similarity searching. Since no collection of this scale was<br />\navailable for research purpose, we had to tackle the non-trivial process of<br />\nimage crawling and descriptive feature extraction (we used five MPEG-7<br />\nfeatures) using the European EGEE computer GRID. The result of this<br />\neffort is a test collection, the first of such scale, that will be opened to the<br />\nresearch community for experiments and comparisons.</p>\n</blockquote>\n
