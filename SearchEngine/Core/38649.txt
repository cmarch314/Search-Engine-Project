HREF=\"Class_Voronoi_intersection_2_traits_3.html#Cross_link_anchor_1133\">Voronoi_intersection_2_traits_3</A>&lt;K&gt;</I> is a model for the concept\n  <I><A It can be used to instantiate the\n  geometric traits class of a two-dimensional regular triangulation.\n  A three-dimensional plane is defined by a point and a vector that\n  are members of the traits class. The triangulation is defined on <MATH><I>3D</I></MATH>\n  points. It is the regular triangulation of the input points\n  projected onto the plane and each weighted with the negative squared\n  distance of the input point to the plane. It can be shown that it is\n  dual to the power diagram obtained by intersecting the\n  three-dimensional Voronoi diagram of the input points with the\n  plane. All predicates and constructions used in the computation of\n  the regular triangulation are formulated on the three dimensional\n  points without explicitly constructing the projected points and the\n  weights. This reduces the arithmetic demands. The traits class is\n  templated by a kernel class <I>K</I>.\n<P>\n\n<I>#include &lt;<A
