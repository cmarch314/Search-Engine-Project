OEBinary Version 2 Format</A>\n</H1>\n\n<P>\nThe OEBinary version 2 format is general format designed to support\nrapid compression, storage and access of numerous data types.  In\ncontrast to OEBinary version 1, it is not specific to molecules or\nmolecular data.  OEBinary version 2 is similar to Extensible Markup\nLanguage (XML) in that any arbitrary data can be stored and identified\nby associated data tags.  Data tag identifiers are used to define the\ndata contents and can be used to decide how the data should be parsed\nand interpreted.  The format supports unlimited levels of recursive\nembedding of data records within data records.  Objects can be made\npersistent across data files and streams through the use of OEBinary\nformat files.
