<I>first</I>...<I>last</I>.\nBeing inside is defined by the odd-even rule. If we take a ray starting at the\npoint and extending to infinity (in any direction), we count the number of\nintersections. If this number is odd, the point is inside, otherwise it is\noutside. If the point is on a polygon edge, a special value is returned.  A\nsimple polygon divides the plane in an unbounded and a bounded region.\nAccording to the definition points in the bounded region are inside the polygon.\n<P>\n\n<H3>Requirements</H3>\n\n   \n
