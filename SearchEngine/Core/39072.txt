By making each\nunit in the input co-ordinates equivalent to 25 pixels in the final\nimage.  By adding suitable borders this determines the default image\nsize.  OEChem's depiction algorithm attempt to make each bond\nlength 1.0, so typically each bond will end up being 25 pixels long.\n\n<P>\nThe entire image can be scaled/resized by using the <code>-scale</code>\ncommand line option.  The default scale (25 pixels per unit) is\ndefined to be 1.0, by the bitmap may be zoomed or shrunk by specifying\na floating point value.  For example, the option <code>-scale 0.75</code>\nshould generate an image a three quarters of the default size, and\n<code>-scale 1.5</code> should generate an image one and a half times the\ndefault.\n\n<P>\nIf both the <code>-wide</code> and <code>-high</code> command line options are\nused to specify a width and height in pixels, respectively (and the\nuser hasn't specified <code>-noauto</code>), the molecule is scaled to\nmaximally fit the given image dimensions.\n\n<P>\nOtherwise, if the user specifies just the width, or just the height,\nor both but in conjunction with <code>-noauto</code>, these values no only\nset the final image dimensions, but also shrink the scale so that\nthe molecule fits that dimension.  Note, that these values are not\nused to zoom a molecule, specify a width and/or height larger than\nthat required by the current scale, causes the molecule to float in\nthe middle of the image of the requested size.\n\n<P>\n\n<DIV CLASS=\"navigation\">\n<p><hr>\n<table align=\"center\" border=\"0\" width=\"100%\" cellpadding=\"0\" cellspacing=\"0\">\n<tr>\n<td rowspan=2><A HREF=\"node9.html\"><img src=\"../icons/previous.gif\"\n  border=\"0\" alt=\"Previous Page\"></A></td>\n<td rowspan=2><A HREF=\"node7.html\"><img src=\"../icons/up.gif\"\n  border=\"0\" alt=\"Up One Level\"></A></td>\n<td rowspan=2><A HREF=\"node11.html\"><img src=\"../icons/next.gif\"\n  border=\"0\" alt=\"Next Page\"></A></td><td width=\"100%\" rowspan=\"2\"> </td>\n<td valign=\"center\" align=\"right\" height=\"56\" background=\"../icons/title_bkgd.gif\" colspan=4 class=\"navigation\">Ogham</td></tr>\n<tr><td><img src=\"../icons/spacer_1.gif\"\n  border=\"0\" alt=\"\"></td>\n<td><A href=\"contents.html\"><img src=\"../icons/contents.gif\"\n  border=\"0\" alt=\"Contents\"></A></td>\n<td><img src=\"../icons/spacer_2.gif\"\n  border=\"0\" alt=\"\"></td>\n<td><img src=\"../icons/blank_4.gif\"\n  border=\"0\" alt=\"\"></td>\n</tr></table>\n<b class=\"navlabel\">Previous:</b> <a class=\"sectref\" HREF=\"node9.html\">3.2 Command Line Options</A>\n<b class=\"navlabel\">Up:</b> <a class=\"sectref\" HREF=\"node7.html\">3.
