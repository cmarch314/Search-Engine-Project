The\nunsigned integer type argument must be constant listed in the\nOEAtomStereo namespace.  If an atom has associated stereochemistry\ndata, it can be retrieved using the OEAtomBase member function\n<code>GetStereo(const std::vector&lt;OEAtomBase*&gt;&amp;,unsigned int)</code> function.\nMultiple possible values may be associated with each class of\nstereochemistry.  For instance, if an atom has associated tetrahedral\nstereochemistry, the possible values are Undefined, RightHanded (or\njust Right), or LeftHanded (or just Left).  The following code sample\ndemonstrates looping over atoms, testing for atoms which have\ntetrahedral stereochemistry, and printing out the value of the\ntetrahedral stereochemistry.\n\n<P>\n<dl><dd><pre class=\"verbatim\">#include \"oechem.h\"\n#include &lt;iostream&gt;\n#include &lt;vector&gt;\n\nusing namespace OESystem;\nusing namespace OEChem;\nusing namespace std;\n\nint main()\n{\n  OEMol mol;\n  OEParseSmiles(mol, \"F[C@H](Cl)Br\");\n\n  OEIter&lt;OEAtomBase&gt; atom,nbr;\n  for (atom=mol.GetAtoms();atom;++atom)\n  {\n    if (atom-&gt;HasStereoSpecified(OEAtomStereo::Tetrahedral))\n    {\n      std::vector&lt;OEAtomBase*&gt; v;\n      for (nbr = atom-&gt;GetAtoms();nbr;++nbr)\n        v.push_back(nbr);\n\n      unsigned int stereovalue =\n        atom-&gt;GetStereo(v,OEAtomStereo::Tetrahedral);\n\n      cerr &lt;&#60; \"Atom: \" &lt;&#60; atom-&gt;GetIdx() &lt;&#60; ' ';\n\n      if (stereovalue == OEAtomStereo::RightHanded)\n        cerr &lt;&#60; \"Right Handed\" &lt;&#60; endl;\n      else if (stereovalue == OEAtomStereo::LeftHanded)\n        cerr &lt;&#60; \"Left Handed\" &lt;&#60; endl;\n      }\n  }\n  return 0;\n}\n</pre>\n<div class=\"verbatiminput-footer\">\n<a href=\"ch14-1.txt\" type=\"text/plain\">Download as text.</a>\n</div>\n</dd></dl>\n\n<P>\nThe definition of <i>handedness</i> for tetrahedral stereochemistry does\nnot imply chirality around a tetrahedral center, but rather indicates\nrelative positions of neighboring atoms.  Note that the function\n<code>GetStereo()</code> requires a STL vector containing pointers to the\nneighboring atoms as the first argument.  The <i>handedness</i> value\nreturned from <code>GetStereo()</code> will depend on the order of the neighboring\natoms as they appear in the vector passed to <code>GetStereo()</code>.  The\ndefinition <i>handedness</i> in OEChem is demonstrated pictorially in\nFigure 14-2.\n\n<P>\nFigure 14-2\n\n<P>\nLooking down the bond between atom number one and the central atom,\n<i>handedness</i> is defined as the direction of travel from atom number\ntwo to atom number three.  The direction of travel must always be\nalong the acute angle formed by atom two, the central atom, and atom\nthree.  Right handed and left handed directions of travel can also be\nthough of as clockwise and counterclockwise, respectively.  The first\nneighbor atom in the vector passed to <code>GetStereo()</code> is taken as atom\nnumber one determination of <i>handedness</i>.  Likewise, subsequent atoms\nin the neighbor atom vector are assigned sequentially to positions in\nthe <i>handedness</i> definition.  Although, three neighboring atoms are\nsufficient to determine the ``handedness'' around a trigonal pyramidal\nor tetrahedral center, either three or four atoms can be provided to\nthe <code>GetStereo()</code> function when requesting a value for tetrahedral\nchirality.\n\n<P>\nSetting the relative stereochemistry around a particular center is\naccomplished using the function <code>SetStereo(const\nstd::vector&lt;OEAtomBase*&gt;&amp;,unsigned int type,unsigned int value)</code>.\nJust as in <code>GetStereo()</code>, the STL vector of neighbor atoms provide\nthe references about which the <i>handedness</i> is defined.  The first\nof the unsigned integer arguments is the stereochemistry type\n(<I>i.e.</I> <code>OEAtomStereo::Tetra</code>), and the second is the associated\nvalue (<I>i.e.</I> <code>OEAtomStereo::Right</code>).\n\n<P>\n\n<DIV CLASS=\"navigation\">\n<p><hr>\n<table align=\"center\" border=\"0\" width=\"100%\" cellpadding=\"0\" cellspacing=\"0\">\n<tr>\n<td rowspan=2><A HREF=\"node132.html\"><img src=\"../icons/previous.gif\"\n  border=\"0\" alt=\"Previous Page\"></A></td>\n<td rowspan=2><A HREF=\"node132.html\"><img src=\"../icons/up.gif\"\n  border=\"0\" alt=\"Up One Level\"></A></td>\n<td rowspan=2><A HREF=\"node134.html\"><img src=\"../icons/next.gif\"\n  border=\"0\" alt=\"Next Page\"></A></td><td width=\"100%\" rowspan=\"2\"> </td>\n<td valign=\"center\" align=\"right\" height=\"56\" background=\"../icons/title_bkgd.gif\" colspan=4 class=\"navigation\">OEChem - C++ Theory Manual\n<BR>\nVersion 1.3.1</td></tr>\n<tr><td><img src=\"../icons/spacer_1.gif\"\n  border=\"0\" alt=\"\"></td>\n<td><A href=\"contents.html\"><img src=\"../icons/contents.gif\"\n  border=\"0\" alt=\"Contents\"></A></td>\n<td><img src=\"../icons/spacer_2.gif\"\n  border=\"0\" alt=\"\"></td>\n<td><img src=\"../icons/blank_4.gif\"\n  border=\"0\" alt=\"\"></td>\n</tr></table>\n<b class=\"navlabel\">Previous:</b> <a class=\"sectref\" HREF=\"node132.html\">14.
