If you're using a different version, the naming convention may differ, but the settings will remain the same.</em>\n</p>\n<ol>\n<li>Start up Thunderbird for the first time (or go to Tools, Account Settings, Account Actions, Add Mail Account)</li>\n<li>In the <b>Mail Account Setup</b> window, enter your name and email address. Enter your password here if you want to have it remembered. Click <strong>Continue</strong>.</li>\n<li>Thunderbird will now attempt to look for the IMAP information. Please make sure that you have <strong>IMAP </strong>selected.</li>\n<li>Click on <b>Manual config </b>in the lower left corner to make sure that the information is as follow:\n<ul>\n<li>Incoming\n<ul>\n<li>IMAP</li>\n<li>Server hostname: <strong>imap.ics.uci.edu</strong></li>\n<li>Port: <strong>993</strong></li>\n<li>SSL: <strong>SSL/TLS</strong></li>\n<li>Authentication: <strong>Normal password</strong></li>\n</ul>\n</li>\n<li>Outgoing\n<ul>\n<li><a href=\"http://www.oit.uci.edu/email/tbird3-IMAP-pc.html\">UCI SMTP</a>.&#160;If you're connecting from a laptop or from off campus, please use the UCI SMTP server settings or your ISP server settings. <br/>\n<ul>\n<li>For UCI SMTP, see settings from step number 7 in the link provided.</li>\n</ul>\n</li>\n<li>ICS SMTP. If your machine is in on the UCI-ICS domain, please use the following settings.&#160;\n<ul>\n<li>\n</li>\n<li>Server hostname: <strong>smtp.ics.uci.edu</strong></li>\n<li>Port: <strong>587</strong></li>\n<li>SSL: <strong>STARTTLS</strong></li>\n<li>Authentication: <strong>Normal password</strong></li>\n<li>Username:<strong> &lt;your username&gt;</strong>\n<ul>\n</ul>\n</li>\n</ul>\n</li>\n</ul>\n</li>\n</ul>\n<br/><img alt=\"advanced window\" src=\"../bin/img/email/setup.jpg\"/>\n</li>\n<li>Click <strong>Re-test</strong> and then <strong>Done</strong></li>\n<li>If you <em>did not</em> choose to have your password saved, you will now be prompted to enter your email password.</li>\n<li>Go to <strong>Tools </strong>and select <strong>Account Settings</strong></li>\n<li>Under <strong>Server Settings</strong>, click on the <b>Advanced</b>&#160;button\n<ul>\n<li>Make sure that <b>IMAP server directory</b> is empty</li>\n<li>Uncheck <b>Show only subscribed folders</b></li>\n<li>Uncheck <b>Allow server to override these namespaces</b></li>\n</ul>\n<br/><img alt=\"advanced window\" src=\"../bin/img/email/advanced.jpg\"/>\n</li>\n<li>This last step is optional.
