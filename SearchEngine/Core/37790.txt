The following example shows how to create\nthe molecule water.\n\n<P>\nAtoms are created by calling the OEMolBase method, <code>NewAtom</code>, and\nBonds are created by calling the OEMolBase method, <code>NewBond</code>.\nOEMolBase NewAtom takes the atomic number of the atom to create and\nreturns a pointer to the new OEAtomBase, and NewBond takes two OEAtomBases\nand a integer bond order as arguments, and returns a pointer to the new\nOEBondBase.\n\n<P>\nThe atoms and bonds of a molecule are automatically deleted when their\nparent molecule is destroyed.\n\n<P>\n<dl><dd><pre class=\"verbatim\">#!/usr/bin/env python\n# ch10-1.py\nfrom openeye.oechem import *\n\nmol = OEGraphMol()\n\no  = mol.NewAtom(8)\nh1 = mol.NewAtom(1)\nh2 = mol.NewAtom(1)\n\nb1 = mol.NewBond(o,h1,1)\nb2 = mol.NewBond(o,h2,1)\n\nprint mol.NumAtoms()\nprint mol.NumBonds()\n</pre>\n<div class=\"verbatiminput-footer\">\n<a href=\"ch10-1.txt\" type=\"text/plain\">Download as text.</a>\n</div>\n</dd></dl>\n\n<P>\nIn the example source code, the atomic numbers of oxygen, 8, and\nhydrogen, 1, are explicitly encoded in the program. This defines the atomic symbols\nin the C++ namespace OEElemNo with the appropriate values.  The\nfollowing example uses these constants instead of just numbers.\n\n<P>\n<dl><dd><pre class=\"verbatim\">#!/usr/bin/env python\n# ch10-2.py\nfrom openeye.oechem import *\n\nmol = OEGraphMol()\n\no  = mol.NewAtom(OEElemNo_O)\nh1 = mol.NewAtom(OEElemNo_H)\nh2 = mol.NewAtom(OEElemNo_H)\n\nb1 = mol.NewBond(o,h1,1)\nb2 = mol.NewBond(o,h2,1)\n\nprint mol.NumAtoms()\nprint mol.NumBonds()\n</pre>\n<div class=\"verbatiminput-footer\">\n<a href=\"ch10-2.txt\" type=\"text/plain\">Download as text.</a>\n</div>\n</dd></dl>\n\n<P>\n\n<DIV CLASS=\"navigation\">\n<p><hr>\n<table align=\"center\" border=\"0\" width=\"100%\" cellpadding=\"0\" cellspacing=\"0\">\n<tr>\n<td rowspan=2><A HREF=\"node104.html\"><img src=\"../icons/previous.gif\"\n  border=\"0\" alt=\"Previous Page\"></A></td>\n<td rowspan=2><A HREF=\"node104.html\"><img src=\"../icons/up.gif\"\n  border=\"0\" alt=\"Up One Level\"></A></td>\n<td rowspan=2><A HREF=\"node106.html\"><img src=\"../icons/next.gif\"\n  border=\"0\" alt=\"Next Page\"></A></td><td width=\"100%\" rowspan=\"2\"> </td>\n<td valign=\"center\" align=\"right\" height=\"56\" background=\"../icons/title_bkgd.gif\" colspan=4 class=\"navigation\">OEChem - Python Theory Manual\n<BR>\nVersion 1.3.1</td></tr>\n<tr><td><img src=\"../icons/spacer_1.gif\"\n  border=\"0\" alt=\"\"></td>\n<td><A href=\"contents.html\"><img src=\"../icons/contents.gif\"\n  border=\"0\" alt=\"Contents\"></A></td>\n<td><img src=\"../icons/spacer_2.gif\"\n  border=\"0\" alt=\"\"></td>\n<td><A href=\"genindex.html\"><img src=\"../icons/index.gif\"\n  border=\"0\" alt=\"Index\"></A></td>\n</tr></table>\n<b class=\"navlabel\">Previous:</b> <a class=\"sectref\" HREF=\"node104.html\">10.
