Recognition</div>\n    <div id=\"paper_authors\">P.&nbsp;Gehler, A.&nbsp;Holub, <a href=\"/people/19.html\">Max&nbsp;Welling</a></div>\n    <div id=\"abstract\">\n      <img style=\"float:left;\" src=\"icon_drop.jpg\" alt=\"icon\">\n      Probabilistic modelling of text data in the bag-of-words representation has\nbeen dominated by directed graphical models such as pLSI, LDA, NMF, and\ndiscrete PCA. Recently, state of the art performance on visual object\nrecognition has also been reported using variants of these models. We\nintroduce an alternative undirected graphical model suitable for modelling\ncount data. This Rate Adapting Poisson (RAP) model is shown to generate\nsuperior dimensionally reduced representations for subsequent retrieval or\nclassification. Models are trained using contrastive divergence while\ninference of latent topical representations is efficiently achieved through a\nsimple matrix multiplication. \n\n\n    </div>\n    <div id=\"bibtext\">\n      <h3>Download: <a href=\"/papers/GehlerHW_ICML_2006/GehlerHW_ICML_2006.pdf\">pdf</a></h3>\n      <h3>Text Reference</h3>\nP.&nbsp;V.
