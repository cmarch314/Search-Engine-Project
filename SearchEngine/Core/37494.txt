Panel-->\n\n<H2><A NAME=\"SECTION0017210000000000000000\">\n15.2.1 OETriposAtomTypes</A>\n</H2>\n\n<P>\nThis function sets the integer atom type field of each atom in a molecule to\nits Tripos atom type.  This function typically requires OEAssignAromaticFlags\nhas been called to perceive aromaticity using the Tripos aromaticity model,\ni.e. <code>OEAssignAromaticFlags(mol,OEAroModelTripos)</code>.  This is required as\nTripos considers compounds such as pyrole, to be aliphatic and so the carbon\natoms of pyrole should be correctly typed as \"C.2\" not \"C.ar\".  Using the\nTripos aromaticity model, however, is not a strong requirement and other\naromaticity models can be used (for example if C.ar is desired in pyrole).\n\n<P>\nThe integer atom type properties are set to the values defined in the\nOETriposType namespace.  So for example, planar nitrogen with Tripos\natom type N.pl3 will be assigned the value OETriposType_Npl3.  Atoms\nare cannot be assigned a Tripos atom type, are assigned value zero,\na.k.a. OETriposType_Du. This function returns true if all the atoms\ncould be assigned at type.\n\n<P>\nTripos integer atom types are required by the Gasteiger partial charge\ncalculation functions.\n\n<P>\n\n<DIV CLASS=\"navigation\">\n<p><hr>\n<table align=\"center\" border=\"0\" width=\"100%\" cellpadding=\"0\" cellspacing=\"0\">\n<tr>\n<td rowspan=2><A HREF=\"node133.html\"><img src=\"../icons/previous.gif\"\n  border=\"0\" alt=\"Previous Page\"></A></td>\n<td rowspan=2><A HREF=\"node133.html\"><img src=\"../icons/up.gif\"\n  border=\"0\" alt=\"Up One Level\"></A></td>\n<td rowspan=2><A HREF=\"node135.html\"><img src=\"../icons/next.gif\"\n  border=\"0\" alt=\"Next Page\"></A></td><td width=\"100%\" rowspan=\"2\"> </td>\n<td valign=\"center\" align=\"right\" height=\"56\" background=\"../icons/title_bkgd.gif\" colspan=4 class=\"navigation\">OEChem - Python Theory Manual\n<BR>\nVersion 1.3.1</td></tr>\n<tr><td><img src=\"../icons/spacer_1.gif\"\n  border=\"0\" alt=\"\"></td>\n<td><A href=\"contents.html\"><img src=\"../icons/contents.gif\"\n  border=\"0\" alt=\"Contents\"></A></td>\n<td><img src=\"../icons/spacer_2.gif\"\n  border=\"0\" alt=\"\"></td>\n<td><A href=\"genindex.html\"><img src=\"../icons/index.gif\"\n  border=\"0\" alt=\"Index\"></A></td>\n</tr></table>\n<b class=\"navlabel\">Previous:</b> <a class=\"sectref\" HREF=\"node133.html\">15.2 Tripos Atom Typing</A>\n<b class=\"navlabel\">Up:</b> <a class=\"sectref\" HREF=\"node133.html\">15.2 Tripos Atom Typing</A>\n<b class=\"navlabel\">Next:</b> <a class=\"sectref\" HREF=\"node135.html\">15.2.2 OETriposAtomTypeNames</A>\n<hr>\n<span class=\"release-info\">Documentation released on July 30, 2004.</span>\n</DIV>\n<!--End of Navigation Panel-->\n\n</BODY>\n</HTML>\n", "id": 37494.0}
