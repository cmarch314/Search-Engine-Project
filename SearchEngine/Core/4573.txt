PL241-Compiler is an SSA-based optimizing compiler that supports copy propagation, common subexpression elimination, constant folding, register allocation and code generation for the DLX (pronounced \u2018Deluxe\u2019) RISC processor architecture.</p>\n\n<p>I wrote the compiler as a part of a graduate course at UC Irvine. It was both a challenging and extremely useful course and compiler to work on. In fact, I ended up borrowing ideas like value-numbering that are pivotal to the SSA-form for my own research projects. I would highly recommend this course to anyone who might be interested in, or is working on program analysis.</p>\n\n<p>Essentially, I decided to open source the code of my implementation and continue working on it during my free time \u2014 yes, it is that interesting and addictive!</p>\n\n<p>Code: <a href=\"https://bitbucket.org/vpalepu/pl241-compiler\">https://bitbucket.org/vpalepu/pl241-compiler</a></p>\n\n</div>\n\n<div><i>TL;DR</i> - SSA-based optimizing compiler that supports copy propagation, common subexpression elimination. </div>\n\n<div style=\"margin-bottom: 10px\">\n\t<br>\n\t<b class=\"label\">Tags:</b>\n\t\n\t\t<span class=\"tags\">compiler</span>\n\t\t\n\t\t\t<span> </span>\n\t\t\n\t\n\t\t<span class=\"tags\">SSA</span>\n\t\t\n\t\t\t<span> </span>\n\t\t\n\t\n\t\t<span class=\"tags\">optimization</span>\n\t\t\n\t\t\t<span> </span>\n\t\t\n\t\n\t\t<span class=\"tags\">code-generation</span>\n\t\t\n\t\t\t<span> </span>\n\t\t\n\t\n\t\t<span class=\"tags\">program-analysis</span>\n\t\t\n\t\n</div>\n</div>\n\n<div class=\"plain stretch\">\n  <h2>Related Posts</h2>\n  <ul class=\"posts\">\n    \n      <li><span>11 Jan 2015</span> &raquo;
