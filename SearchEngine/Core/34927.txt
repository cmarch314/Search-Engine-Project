to uniquely identify a position in the document:\n</p>\n<pre class=\"wiki\">[=#point1]\n</pre><p>\nThis syntax was chosen to match the format for explicitly naming the header id <a class=\"wiki\" For example:\n</p>\n<pre class=\"wiki\">== Long title == #title\n</pre><p>\nIt's also very close to the syntax for the corresponding link to that anchor:\n</p>\n<pre id=\"Images\">Images</h2>\n<p>\nUrls ending with <tt>.png</tt>, <tt>.gif</tt> or <tt>.jpg</tt> are no longer automatically interpreted as image links, and converted to <tt>&lt;img&gt;</tt> tags.\n</p>\n<p>\nYou now have to use the [[Image]] macro.
