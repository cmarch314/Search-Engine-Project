Navigation Panel-->\n\n<H2><A NAME=\"SECTION007350000000000000000\">\n5.3.5 SD Data Example</A>\n</H2>\n\n<P>\nThe next example will show how to use the tagged data methods.\n\n<P>\n<dl><dd><pre class=\"verbatim\">#!/usr/bin/env python\n# ch5-2.py\nfrom openeye.oechem import *\nimport os, sys\n\nmol = OEGraphMol()\nOEParseSmiles(mol, \"c1ccccc1\")\nmol.SetTitle(\"benzene\")\n\n# now set some tagged data\nOESetSDData(mol, 'color', 'brown')\nOESetSDData(mol, 'size', 'small')\nOESetSDData(mol, 'natoms', str(mol.NumAtoms()))\n\n# loop over data and print it out\nfor dp in OEGetSDDataPairs(mol):\n   sys.stdout.write('%s : %s&#92;n' % (dp.GetTag(), dp.GetValue()))\n\n# check for existence of a field, then delete it\nif OEHasSDData(mol, 'color') == 1:\n    OEDeleteSDData(mol, 'color')\n\n# one last loop shows no 'color' field\nfor dp in OEGetSDDataPairs(mol):\n    sys.stdout.write('%s : %s&#92;n' % (dp.GetTag(), dp.GetValue()))\n</pre>\n<div The other, <code>mergecsv.py</code>, takes a csv file and\nadds the data as tags to molecules in an input stream. This simple\nscript assumes that the first column is the molecule title matching\ntitles found in the incoming molecule file.
