to download the MSDNAA software once\n          your ICS account expires.</p>\n<a id=\"download\" name=\"download\"></a> <b>Q: How can I obtain the software?</b>\n<p> You may download the software ISO images\n          of the software in the \"MSDNAA Software\" folder. &#160;You can burn your own\n          CDs/DVDs to take home or bring a USB flash drive and transfer the ISO files you might need. &#160;However, you will still need to login into \n          MSDNAA site and \"purchase\" the software from their website\n          to get the necessary product keys. </p>\n<a id=\"problems\" name=\"problems\"></a><b>Q: Why is the size of the file I downloaded so small or unexecutable?</b>\n<p>If you are downloading the MSDNAA software, the file that first gets\n          downloaded may &#160;It is a downloader file of a very small size\n          that is used to download an SDC file. &#160;After the SDC file is downloaded,\n          the downloader is required to extract the actual completed. &#160;You need to use the Downloader to extract the\n          install files from the SDC file. &#160;The SDC file is safe to delete after the setup files are extracted from it. </p>\n<a
