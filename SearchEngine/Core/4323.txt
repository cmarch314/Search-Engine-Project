align=\"center\">\n<h1>The FLAMINGO Project on Data Cleaning</h1>\n<h3>Department of Computer Science, UC Irvine</h3>\n</table>\n\n<h2>Objective</h2>\n\n<p> The Flamingo Project focuses on <b>data cleaning</b>, i.e., how to\ndeal with errors and inconsistencies in information systems.  As an\nexample, in many applications such as data integration, commercial\norganizations need to collect data from various sources to conduct\nanalysis and make decisions. Often, the data from these different\nsources can have inconsistencies. For instance, we use first name,\nlast name, SSN, and birthday to identify a person. However, the same\nname, e.g., \"Schwarzenegger\", may be misspelled as \"Swarzzengaer\" or\nother forms. Such errors make it more challenging to link records from\ndifferent places and answer queries approximately.  We are developing\nalgorithms in order to make query answering and information retrieval\nefficient in the presence of such inconsistencies and errors.<p>\n\nWith the NSF\naward <a\nhref=\"http://www.nsf.gov/awardsearch/showAward.do?AwardNumber=0844574\">IIS-0844574</a>,\nwe plan to study the following problems.  Supporting fuzzy queries is\nbecoming increasingly more important in applications that need to deal\nwith a variety of data inconsistencies in structures, representations,\nor semantics.  Many existing algorithms require an offline analysis of\ndata sets to construct an efficient index structure to support online\nquery processing. Fuzzy join queries of data sets are more time\nconsuming due to the computational complexity.  The PI is studying\nthree research problems: (1) constructing high-quality inverted lists\nfor fuzzy search queries using Hadoop; (2) supporting fuzzy joins of\nlarge data sets using Hadoop; and (3) using the developed techniques\nto improve data quality of large collections of documents.<p>\n\nWith the NSF\naward <a\nhref=\"http://www.nsf.gov/awardsearch/showAward.do?AwardNumber=1030002\">1030002</a>,\nwe will study how to support powerful keyword search with efficient\nindexing structures and algorithms in a clouding-computing\ninfrastructure. A main application is supporting\n<a href=\"http://fr.ics.uci.edu/\">family reunification</a> in disasters\nsuch as the Haiti Earthquake.  Check our portals for\nthe <a href=\"http://fr.ics.uci.edu/haiti/\">Haiti Earthquake</a>\nand <a href=\"http://fr.ics.uci.edu/chile/\">Chile Earthquake</a>.  The\nmain challenge is how to use limited programming primitives in the\ncloud to implement index structures and search algorithms. \n\n<h2>Our <a href=\"spellchecker/\">qSpeller project page</a> for the <a href=\"http://web-ngram.research.microsoft.com/spellerchallenge/\">Microsoft Speller Challenge</a>.</h2>\n\n<h2>News</h2>\n\n<ul>\n\n<li>(1/13/2013) Our DASFAA 2003 paper titled \"Efficient Record Linkage\nin Large Data Sets\" received the 10-year Best Paper Award for DASFAA\n2013.
