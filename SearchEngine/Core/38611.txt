Reactions</A>\n</H1>\n\n<P>\nReaction processing in OEChem is divided into two categories:\nunimolecular reactions and library generation.  Unimolecular reactions\nare useful for (although not limited to) normalization reactions. The\n<code>OEUniMolecularRxn</code> class in OEChem applies chemical\ntransformations to individual molecules.  Reactions can also be used\nto generate combinatorial libraries using OEChem.  Both 'clipping' and\nreaction based enumeration can be achieved using the\n<code>OELibraryGen</code> class.\n\n<P>\nReactions are represented in OEChem as query molecules\n(<code>OEQMolBase</code>).  Sets of chemical transform operations are\nderived from reaction molecules by differences between the reactant\nand product patterns and in the reaction molecule.  For example, atoms\nand bonds that appear in the reactant pattern, but are absent the in\nthe product pattern are 'destroyed'.  Atoms and bonds that appear in\nthe product pattern but not in the reactant pattern are 'created'.\nAtoms are tracked between reactants and products by means atom maps.\nAtom maps are stored and retrieved using the\n<code>OEAtomBase::SetMapIdx</code> and <code>OEAtomBase::GetMapIdx</code> methods.\nProduct atoms that have the same map index as reactant atoms originate\nfrom their reactant counterpart.  Reactions are completely defined by\na fields in a query molecule.  Reaction molecules can be created from\nvirtually any reaction file format (SMIRKS, MDL RXN, etc.), and can\neven be constructed programmatically.\n\n<P>\n\n<p><hr>\n<!--Table of Child-Links-->\n<A NAME=\"CHILD_LINKS\"><STRONG>Subsections</STRONG></a>\n\n<UL CLASS=\"ChildLinks\">\n<LI><A href=\"node248.html\">30.1 Normalization Reactions</a>\n<LI><A href=\"node249.html\">30.2 Library Generation</a>\n</ul>\n<!--End of Table of Child-Links-->\n\n<DIV CLASS=\"navigation\">\n<p><hr>\n<table align=\"center\" border=\"0\" width=\"100%\" cellpadding=\"0\" cellspacing=\"0\">\n<tr>\n<td rowspan=2><A HREF=\"node246.html\"><img src=\"../icons/previous.gif\"\n  border=\"0\" alt=\"Previous Page\"></A></td>\n<td rowspan=2><A HREF=\"cplusprog.html\"><img src=\"../icons/up.gif\"\n  border=\"0\" alt=\"Up One Level\"></A></td>\n<td rowspan=2><A HREF=\"node248.html\"><img src=\"../icons/next.gif\"\n  border=\"0\" alt=\"Next Page\"></A></td><td width=\"100%\" rowspan=\"2\"> </td>\n<td valign=\"center\" align=\"right\" height=\"56\" background=\"../icons/title_bkgd.gif\" colspan=4 class=\"navigation\">OEChem - C++ Theory Manual\n<BR>\nVersion 1.3.1</td></tr>\n<tr><td><img src=\"../icons/spacer_1.gif\"\n  border=\"0\" alt=\"\"></td>\n<td><A href=\"contents.html\"><img src=\"../icons/contents.gif\"\n  border=\"0\" alt=\"Contents\"></A></td>\n<td><img src=\"../icons/spacer_2.gif\"\n  border=\"0\" alt=\"\"></td>\n<td><img src=\"../icons/blank_4.gif\"\n  border=\"0\" alt=\"\"></td>\n</tr></table>\n<b class=\"navlabel\">Previous:</b> <a class=\"sectref\" HREF=\"node246.html\">29.1.3 Logical Operators</A>\n<b class=\"navlabel\">Up:</b> <a class=\"sectref\" HREF=\"cplusprog.html\">OEChem - C++ Theory</A>\n<b class=\"navlabel\">Next:</b> <a class=\"sectref\" HREF=\"node248.html\">30.1 Normalization Reactions</A>\n<hr>\n<span class=\"release-info\">Documentation released on July 30, 2004.</span>\n</DIV>\n<!--End of Navigation Panel-->\n\n</BODY>\n</HTML>\n", "id": 38611.0}
