  <div class=\"wikipage searchable\">\n        \n          <div id=\"wikipage\" class=\"trac-content\"><h2 id=\"HowisdevelopmentdoneonNomaticIM\">How is development done on Nomatic*IM?</h2>\n<p>\nNomatic*IM is a research project at UCI.  Our current structure is a benevolent dictatorship run by Professor Patterson.  The development goals are largely set by our research agenda.  We are eager to see broad adoption of our program however and to this end are open to hearing from the needs of users/developers.  We are happy to address <a class=\"wiki\" href=\"/LUCICodeRepository/nomaticIM/wiki/BugReport\">bug reports</a>, <a class=\"wiki\" href=\"/LUCICodeRepository/nomaticIM/wiki/Security\">security problems</a>, and <a class=\"missing wiki\">feature requests?</a>. \n</p>\n<h3 id=\"Currentlistofcommitters\">Current list of committers</h3>\n<ol><li>Sharon Ding (dingx at ics.uci.edu)\n</li><li>Sam Kaufman (kaufmans at uci.edu)\n</li><li>Donald J. Patterson (djp3 at ics.uci.edu)\n</li></ol><h2 id=\"RequirementsforDevelopers\">Requirements for Developers</h2>\n<ul><li>an IM client.  The following are supported:\n<ul><li>Pidgin (formerly known as gaim)\n</li><li>Skype\n</li><li>iChat\n</li><li>Adium\n</li><li>Web Broadcast via\n<ul><li>Twitter\n</li></ul></li></ul></li><li>Xcode (verson &gt;= 2.4.1)\n</li><li>GCC (the GNU C Compiler) to work on Pidgin plugin only\n</li><li>Ant (&gt; 1.7.0)\n</li><li>Java 1.5+\n</li></ul><h2 id=\"InstructionstosetupabuildenvironmentinWindows\">Instructions to set up a build environment in Windows</h2>\n<ol><li>Install a java jdk\n</li><li>Install cygwin   \n</li><li>In our .bashrc file in cygwin we set our JAVA_HOME to point to the jdk\n<ol class=\"loweralpha\"><li><tt>export JAVA_HOME=\"/cygdrive/c/Program Files/Java/jdk1.6.0_12\"</tt>\n</li></ol></li><li>Installed the ant binary package for Windows (not in cygwin)\n<ol class=\"loweralpha\"><li>set ANT_HOME to point to the Windows install of ant (in .bashrc in cygwin)\n<ol class=\"loweralpha\"><li><tt>export ANT_HOME=\"/cygdrive/c/ant\"</tt>\n</li></ol></li></ol></li><li>set our PATH to see the ant binary and java binaries  (in .bashrc in cygwin)\n<ol class=\"loweralpha\"><li><tt>export PATH=\"$PATH:/ant/bin:/cygdrive/c/Program Files/Java/jdk1.6.0_12/bin\"</tt>\n</li></ol></li><li>If you are a committer, install TortoiseSVN\n<ol class=\"loweralpha\"><li>the svn that comes with cygwin was causing problems for svn+ssh commaands.  It wasn't a problem for read-only svn.\n</li></ol></li></ol><p>\n    \n</p>\n<h2 id=\"Instructionstobuildandrunthemainservice\">Instructions to build and run the main service</h2>\n<ol><li>Check out the Nomatic*IM trunk out of the repository with the command:\n<ol class=\"loweralpha\"><li><tt>svn co svn://djp3-pc2.ics.uci.edu:3690/nomaticIM/nomatic/trunk nomatic</tt>\n</li></ol></li><li>Change into the new <tt>nomatic</tt> directory\n</li><li>edit the build.local.properties file as necessary\n<ol class=\"loweralpha\"><li>See the examples in the source tree\n</li></ol></li><li>Run the command <tt>ant build</tt> [ or <tt>ant build-mac-app</tt> or <tt>ant build-win-app</tt> or <tt>ant build-linux-app</tt> ]\n</li><li>Run the command <tt>ant run</tt> or [ get the application from the /bin directory ]\n</li></ol><p>\n   \n</p>\n<h2 id=\"Processes\">Processes</h2>\n<ol><li><a class=\"wiki\" href=\"/LUCICodeRepository/nomaticIM/wiki/SubmittingPatch\">Submitting a
