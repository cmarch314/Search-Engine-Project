class=\"False\"><a class=\"False\" href=\"/LUCICodeRepository/nomaticIM/wiki/TracNotification\">Notification</a></li></ul></div><p>\n</p>\n<p>\nThe timeline provides a historic view of the project in a single report.\n</p>\n<p>\nIt lists all Trac events that have occurred in chronological order, a\nbrief description of each event and if applicable, the person responsible for\nthe change.\n</p>\n<p>\nThe timeline lists these kinds of events:\n</p>\n<ul><li><strong>Wiki page events</strong> \u2014 Creation and changes\n</li><li><strong>Ticket events</strong> \u2014 Creation and resolution/closing (and optionally other changes)\n</li><li><strong>Source code changes </strong> \u2014 Repository check-ins\n</li><li><strong>Milestone </strong> \u2014 Milestone completed\n</li></ul><p>\nEach event entry provides a hyperlink to the specific event in question, who authored the change as well as\na brief excerpt of the actual comment or text, if available.\n</p>\n<p>\nIt is possible to filter the displayed events with the various filters in the option panel:\n</p>\n<ul><li><em>View changes from</em> \u2014 the date from which to start displaying events (current date if empty). Events that occurred after this date will not be shown, only those that occurred before that date.\n</li><li><em>and X days back</em> \u2014 how many days backwards in time to get events.\n</li><li><em>done by</em> \u2014 the author of an event. It accepts a space-separated list of authors for which events should be included. Alternatively, if the author names are prefixed by a \"-\" character, then the events having those authors will be excluded, and all the others included. Single or double quotes can be used for specifying author names containing space characters. <em>(since 0.12)</em>\n</li><li><em>Changesets in all repositories</em> \u2014 if you have more than one repository connected to your Trac project, then you can filter the output so events from specific repositories are not shown. <em>(since 0.12)</em>\n</li><li><em>Milestones reached</em> \u2014 display or hide milestones reached.\n</li><li><em>Opened and closed tickets</em> \u2014 display or hide ticket open or close events.\n</li><li><em>Wiki changes</em> \u2014 display or hide Wiki change events.\n</li></ul><p>\nSee TracIni's <a class=\"wiki\" href=\"/LUCICodeRepository/nomaticIM/wiki/TracIni#timeline-section\">[timeline] section</a> for timeline configuration options.\n</p>\n<h2 id=\"RSSSupport\">RSS Support</h2>\n<p>\nThe Timeline module supports subscription using RSS 2.0 syndication. To subscribe to project events, click the orange <strong>XML</strong> icon at the bottom of the page.
