style=\"float:left;\" src=\"icon_drop.jpg\" alt=\"icon\">\n      Learning models for detecting and classifying object categories is a\nchallenging problem in machine vision. While discriminative approaches to\nlearning and classification have, in principle, superior performance,\ngenerative approaches provide many useful features, one of which is the\nability to naturally establish explicit correspondence between model components\nand scene features  this, in turn, allows for the handling of missing data and\nunsupervised learning in clutter. We explore a hybrid generative/discriminative approach using Fisher kernels [1] which retains most of\nthe desirable properties of generative methods, while increasing the\nclassification performance through a discriminative setting. Furthermore, we\ndemonstrate how this kernel framework can be used to combine different types\nof features and models into a single classifier. Our experiments, conducted\non a number of popular benchmarks, show strong performance improvements over\nthe corresponding generative approach and are competitive with the best results\nreported in the literature.\n\n    </div>\n    <div id=\"bibtext\">\n      <h3>Download: <a href=\"/papers/HolubWP_ICCV_2005/HolubWP_ICCV_2005.pdf\">pdf</a></h3>\n      <h3>Text Reference</h3>\nAlex&nbsp;D.
