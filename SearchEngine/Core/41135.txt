RetroSynthesis</h1><br /><br />\n<pre class=\"literalblock\">\nInitialized with a set of reagent objects, generates a retro version\nof each reagent.  Also needs a link to some pool of reactants including\na similarity measure / score between intermediate products found to items\nin the pool.\n\nGiven a desired target structure, will attempt to find a path from starting\ncomponents in the reactant pool to the target via a series of reagent reactions,\nby searching backwards.\n\nGeneral algorithm:\n    While the target mol(s) are not found in the reactant pool (closest item has &lt; 1.0 similarity)\n    and we haven't exceeded the maximum number of steps for consideration:\n        Apply each retro reagent to the target mol and see if it yields any retro products (source reactants).\n        Pick one or more of these retro steps based on how some scoring function of the retro products.\n        Recurse with the retro products being target molecules themselves.\n\nLimitations:\n    - Assumes all retro-reactions expect a single (non-inherent) reactant\n    - If no exact match is found, just returns the last one tried, rather than sum integration over all or even a ranked list\n\n</pre><br /><br />\n\n<!-- ==================== INSTANCE METHODS ==================== -->\n<a name=\"section-InstanceMethods\"></a>\n<table
