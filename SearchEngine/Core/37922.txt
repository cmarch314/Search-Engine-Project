The\ninteger type argument must be a constant listed in the\nOEAtomStereo namespace.  If an atom has associated stereochemistry\ndata, it can be retrieved using the OEAtomBase member function\nGetStereo(const std::vector&lt;OEAtomBase*&gt;&amp;,unsigned int) function.\nMultiple possible values may be associated with each class of\nstereochemistry.  For instance, if an atom has associated tetrahedral\nstereochemistry, the possible values are Undefined, RightHanded (or\njust Right), or LeftHanded (or just Left).  The following code sample\ndemonstrates looping over atoms, testing for atoms which have\ntetrahedral stereochemistry, and printing out the value of the\ntetrahedral stereochemistry.\n\n<P>\n<dl><dd><pre class=\"verbatim\"># ch14-1.py\nfrom openeye.oechem import *\nimport os,sys\n\nmol = OECreateOEMol()\nOEParseSmiles(mol, \"F[C@H](Cl)Br\")\n\nfor atom in mol.GetAtoms():\n    if atom.HasStereoSpecified(OEAtomStereo_Tetrahedral):\n        v= []\n        for nbr in atom.GetAtoms():\n            v.append(nbr)\n\n        stereovalue = atom.GetStereo(v, OEAtomStereo_Tetrahedral)\n\n        print \"Atom:\",atom.GetIdx(),\" \"\n\n        if stereovalue == OEAtomStereo_RightHanded:\n            print \"Right Handed\"\n        elif stereovalue == OEAtomStereo_LeftHanded:\n            print \"Left Handed\"\n</pre></dl>\n\n<P>\nThe definition of ``handedness'' for tetrahedral stereochemistry does\nnot imply chirality around a tetrahedral center, but rather indicates\nrelative positions of neighboring atoms.  Note that the function\nGetStereo() requires a STL vector containing pointers to the\nneighboring atoms as the first argument.  The ``handedness'' value\nreturned from GetStereo() will depend on the order of the neighboring\natoms as they appear in the vector passed to GetStereo().  The\ndefinition ``handedness'' in OEChem is demonstrated pictorially in\nFigure 14-2.\n\n<P>\nFigure 14-2\n\n<P>\nLooking down the bond between atom number one and the central atom,\n<i>handedness</i> is defined as the direction of travel from atom number\ntwo to atom number three.  The direction of travel must always be\nalong the acute angle formed by atom two, the central atom, and atom\nthree.  Right handed and left handed directions of travel can also be\nthough of as clockwise and counterclockwise, respectively.  The first\nneighbor atom in the vector passed to GetStereo() is taken as atom\nnumber one determination of ``handedness''.  Likewise, subsequent atoms\nin the neighbor atom vector are assigned sequentially to positions in\nthe ``handedness'' definition.  Although, three neighboring atoms are\nsufficient to determine the ``handedness'' around a trigonal pyramidal\nor tetrahedral center, either three or four atoms can be provided to\nthe GetStereo() function when requesting a value for tetrahedral\nchirality.\n\n<P>\nSetting the relative stereochemistry around a particular center is\naccomplished using the function SetStereo(AtomVector, type, value).\nJust as in GetStereo(), the AtomVector of neighbor atoms provide the\nreferences about which the <i>handedness</i> is defined.  The first of\nthe unsigned integer arguments is the stereochemistry type (i.e.\nOEAtomStereo_Tetra), and the second is the associated value\n(i.e. OEAtomStereo_Right).\n\n<P>\n\n<DIV CLASS=\"navigation\">\n<p><hr>\n<table align=\"center\" border=\"0\" width=\"100%\" cellpadding=\"0\" cellspacing=\"0\">\n<tr>\n<td rowspan=2><A HREF=\"node128.html\"><img src=\"../icons/previous.gif\"\n  border=\"0\" alt=\"Previous Page\"></A></td>\n<td rowspan=2><A HREF=\"node128.html\"><img src=\"../icons/up.gif\"\n  border=\"0\" alt=\"Up One Level\"></A></td>\n<td rowspan=2><A HREF=\"node130.html\"><img src=\"../icons/next.gif\"\n  border=\"0\" alt=\"Next Page\"></A></td><td width=\"100%\" rowspan=\"2\"> </td>\n<td valign=\"center\" align=\"right\" height=\"56\" background=\"../icons/title_bkgd.gif\" colspan=4 class=\"navigation\">OEChem - Python Theory Manual\n<BR>\nVersion 1.3.1</td></tr>\n<tr><td><img src=\"../icons/spacer_1.gif\"\n  border=\"0\" alt=\"\"></td>\n<td><A href=\"contents.html\"><img src=\"../icons/contents.gif\"\n  border=\"0\" alt=\"Contents\"></A></td>\n<td><img src=\"../icons/spacer_2.gif\"\n  border=\"0\" alt=\"\"></td>\n<td><A href=\"genindex.html\"><img src=\"../icons/index.gif\"\n  border=\"0\" alt=\"Index\"></A></td>\n</tr></table>\n<b class=\"navlabel\">Previous:</b> <a class=\"sectref\" HREF=\"node128.html\">14.
