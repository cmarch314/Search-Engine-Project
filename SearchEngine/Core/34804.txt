searchable\">\n        \n          <div id=\"wikipage\" class=\"trac-content\"><h1 id=\"SupportforInterWikilinks\">Support for <a class=\"wiki\" href=\"/LUCICodeRepository/nomaticIM/wiki/InterWiki\">InterWiki</a> links</h1>\n<p>\n<em>(since <a class=\"ext-link\" href=\"http://trac.edgewall.org/intertrac/milestone%3A0.10\" title=\"milestone:0.10 in Trac project trac\"><span class=\"icon\">\u200b</span>0.10</a>)</em>\n</p>\n<h2 id=\"Definition\">Definition</h2>\n<p>\nAn <a class=\"wiki\" href=\"/LUCICodeRepository/nomaticIM/wiki/InterWiki\">InterWiki</a> link can be used for referring to a Wiki page\nlocated in another Wiki system, and by extension, to any object\nlocated in any other Web application, provided a simple URL \nmapping can be done.\n</p>\n<p>\nAt the extreme, <a class=\"wiki\" href=\"/LUCICodeRepository/nomaticIM/wiki/InterWiki\">InterWiki</a> prefixes can even be used to simply introduce\nlinks to new protocols, such as <tt>tsvn:</tt> used by <a class=\"ext-link\" href=\"http://trac.edgewall.org/intertrac/TortoiseSvn\" title=\"TortoiseSvn in Trac project trac\"><span class=\"icon\">\u200b</span>TortoiseSvn</a>.\n</p>\n<h2 id=\"LinkSyntax\">Link Syntax</h2>\n<pre class=\"wiki\">&lt;target_wiki&gt;(:&lt;identifier&gt;)+\n</pre><p>\nThe link is composed by the targeted Wiki (or system) name,\nfollowed by a colon (e.g. <tt>MeatBall:</tt>),\nfollowed by a page specification in the target.\nNote that, as for <a class=\"wiki\" href=\"/LUCICodeRepository/nomaticIM/wiki/InterTrac\">InterTrac</a> prefixes, <strong><a class=\"wiki\" href=\"/LUCICodeRepository/nomaticIM/wiki/InterWiki\">InterWiki</a> prefixes are case insensitive</strong>.\n</p>\n<p>\nThe target Wiki URL is looked up in the <a class=\"wiki\" href=\"/LUCICodeRepository/nomaticIM/wiki/InterMapTxt\">InterMapTxt</a> wiki page, \nmodelled after <a class=\"ext-link\" href=\"http://www.usemod.com/cgi-bin/mb.pl?InterMapTxt\" title=\"InterMapTxt in MeatBall\"><span class=\"icon\">\u200b</span>MeatBall:InterMapTxt</a>.\n</p>\n<p>\nIn addition to traditional <a class=\"wiki\" href=\"/LUCICodeRepository/nomaticIM/wiki/InterWiki\">InterWiki</a> links, where the target\nis simply <em>appended</em> to the URL, \nTrac supports parametric <a class=\"wiki\" href=\"/LUCICodeRepository/nomaticIM/wiki/InterWiki\">InterWiki</a> URLs:\nidentifiers <tt>$1</tt>, <tt>$2</tt>, ... in the URL\nwill be replaced by corresponding arguments.\nThe argument list is formed by splitting the page identifier\nusing the \":\" separator.\n</p>\n<h2 id=\"Examples\">Examples</h2>\n<p>\nIf the following is an excerpt of the <a class=\"wiki\" href=\"/LUCICodeRepository/nomaticIM/wiki/InterMapTxt\">InterMapTxt</a> page:\n</p>\n<pre class=\"wiki\">= InterMapTxt =\n== This is the place for defining InterWiki prefixes ==\n\nCurrently active prefixes: [[InterWiki]]\n\nThis page is modelled after the MeatBall:InterMapTxt page.\nIn addition, an optional comment is allowed after the mapping.\n----\n{{{\nPEP      http://www.python.org/peps/pep-$1.html           # Python Enhancement Proposal $1 \nTrac-ML  http://thread.gmane.org/gmane.comp.version-control.subversion.trac.general/$1  # Message $1 in Trac Mailing List\n\ntsvn     tsvn:                                            # Interact with TortoiseSvn\n...\nMeatBall http://www.usemod.com/cgi-bin/mb.pl?\nMetaWiki http://sunir.org/apps/meta.pl?\nMetaWikiPedia http://meta.wikipedia.org/wiki/\nMoinMoin http://moinmoin.wikiwikiweb.de/\n...\n}}}\n</pre><p>\nThen, \n</p>\n<ul><li><tt>MoinMoin:InterWikiMap</tt> should be rendered as <a class=\"ext-link\" href=\"http://moinmoin.wikiwikiweb.de/InterWikiMap\" title=\"InterWikiMap in MoinMoin\"><span class=\"icon\">\u200b</span>MoinMoin:InterWikiMap</a>\nand the <em>title</em> for that link would be \"InterWikiMap in MoinMoin\"\n</li><li><tt>Trac-ML:4346</tt> should be rendered as <a class=\"ext-link\" href=\"http://thread.gmane.org/gmane.comp.version-control.subversion.trac.general/4346\" title=\"Message 4346 in Trac Mailing List\"><span class=\"icon\">\u200b</span>Trac-ML:4346</a>\nand the <em>title</em> for that link would be \"Message 4346 in Trac Mailing List\"\n</li></ul><hr />\n<p>\nSee also: <a class=\"wiki\" href=\"/LUCICodeRepository/nomaticIM/wiki/InterTrac\">InterTrac</a>, <a
