The constructor takes 3 arguments, with the third\noptional.\n\n<P>\ndots = OEDots(1000, 50, \"molecules\")\n\n<P>\nThe first argument is how many count events happen on each line of\ndots, the second is how many counts each dot equals. The last argument\nis simply a text label such that at the end of each line, the text\nsays what is being counted. The example above will print a dot (to\nstderr) every 50 events. After 1000 event, it will end the line of\ndots with \"1000 molecules processed, 2000 molecules processed\",\netc. If either of the numeric arguments are set to zero (0), there\nwill be no output but the internal counter will still count. This is\nuseful if you want to count number of times through a loop, but don't\ncare about a status update.\n\n<P>\nIn order up increment the counter, call the Update() method each time\nthrough the loop. If you loop in groups of more than one, you can call\nUpdate(n), where n is a positive integer.\n\n<P>\nAfter the end of the loop, call the Total() member function to get\noutput to stderr of the total number of items counted. The following\nexample shows how to use OEDots:\n\n<P>\n<dl><dd><pre class=\"verbatim\">from openeye.oechem import *\nimport os, sys\n\nifs = oemolistream()\nifs.open(\"nci.smi\")\n\n# create a dots instance\ndots = OEDots(10000,500,\"SMILES\")\n\n# loop over the input file, call update for each molecule\nfor mol in ifs.GetOEGraphMols():\n    DoSomething(mol)\n    dots.Update()\n\n# output a summary\ndots.Total()\n</pre></dl>\n\n<P>\n\n<DIV CLASS=\"navigation\">\n<p><hr>\n<table align=\"center\" border=\"0\" width=\"100%\" cellpadding=\"0\" cellspacing=\"0\">\n<tr>\n<td rowspan=2><A HREF=\"node214.html\"><img src=\"../icons/previous.gif\"\n  border=\"0\" alt=\"Previous Page\"></A></td>\n<td rowspan=2><A HREF=\"node213.html\"><img src=\"../icons/up.gif\"\n  border=\"0\" alt=\"Up One Level\"></A></td>\n<td rowspan=2><A HREF=\"node216.html\"><img src=\"../icons/next.gif\"\n  border=\"0\" alt=\"Next Page\"></A></td><td width=\"100%\" rowspan=\"2\"> </td>\n<td valign=\"center\" align=\"right\" height=\"56\" background=\"../icons/title_bkgd.gif\" colspan=4 class=\"navigation\">OEChem - Python Theory Manual\n<BR>\nVersion 1.3.1</td></tr>\n<tr><td><img src=\"../icons/spacer_1.gif\"\n  border=\"0\" alt=\"\"></td>\n<td><A href=\"contents.html\"><img src=\"../icons/contents.gif\"\n  border=\"0\" alt=\"Contents\"></A></td>\n<td><img src=\"../icons/spacer_2.gif\"\n  border=\"0\" alt=\"\"></td>\n<td><A href=\"genindex.html\"><img src=\"../icons/index.gif\"\n  border=\"0\" alt=\"Index\"></A></td>\n</tr></table>\n<b class=\"navlabel\">Previous:</b> <a class=\"sectref\" HREF=\"node214.html\">23.1 OEStopWatch</A>\n<b class=\"navlabel\">Up:</b> <a class=\"sectref\" HREF=\"node213.html\">23.
