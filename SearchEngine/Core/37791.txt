   <A NAME=\"ccRef_Kernel_d\"></A>\n<h2><I>Kernel_d</I></h2>\n<P>\n\nThe concept of a <I>kernel</I> is defined by a set of requirements on\nthe provision of certain types and access member functions to create \nobjects of these types. The types are function object classes to be used\nwithin the algorithms and data structures in the basic library of C<SMALL>GAL</SMALL>. \nThis allows you to use any model of a kernel as a traits class in \nthe C<SMALL>GAL</SMALL> algorithms and data structures, unless they require types \nbeyond those provided by a kernel.\n<P>\n\n<I><A HREF=\"Concept_Kernel_d.html#Cross_link_anchor_546\">Kernel_d</A></I> subsumes the concept of a <I><MATH><I>d</I></MATH>-dimensional kernel</I>.\n<P>\n\nA kernel provides types, construction objects, and generalized\npredicates.
