This configuration\nincludes the verbose flag, a flag that controls writing a file with or\nwithout comments, etc.  The class is the base class of the classes\n<I><A HREF=\"Class_Pm_file_scanner.html#Cross_link_anchor_807\">Pm_file_scanner</A></I> and <I><A HREF=\"Class_Pm_file_writer.html#Cross_link_anchor_809\">Pm_file_writer</A></I>.  The list of\nparameters for <I><A HREF=\"Class_File_header.html#Cross_link_anchor_805\">File_header</A></I> is as follows:\n<P>\n\n<UL>\n<LI><B>verbose </B> define a verbose format for printing the <I>Planar map</I>. This flag is set by default to false.\n<LI><B>ascii</B> ascii format. This flag is set by default to true.\n<LI><B>no_comments</B> - no comments in output. This flag is set by default to false.\n<LI><B>index_offset</B> - offset for the indices. This flag is set by default to 0.\n<LI><B>has_colors</B> - The format specifies the colors of the <I>Planer map</I> components. This flag is set by default to false.\n<LI><B>dimension</B> - specifies the dimension of <I>Planer map</I>. This flag is set by default to 2.\n</UL>\n<P>\n\nIn addition the <I><A HREF=\"Class_File_header.html#Cross_link_anchor_805\">File_header</A></I> class includes the information regarding the \nnumber
