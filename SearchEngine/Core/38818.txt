The following example shows how to create\nthe molecule water.\n\n<P>\nAtoms are created by calling the OEMolBase method, <code>NewAtom()</code>, and\nBonds are created by calling the OEMolBase method, <code>NewBond()</code>.\n<code>OEMolBase::NewAtom</code> takes the atomic number of the atom to create and\nreturns a pointer to an OEAtomBase, and <code>NewBond</code> takes two OEAtomBase*\nand a integer bond order as arguments, and returns a pointer to an\nOEBondBase result.\n\n<P>\nThe atoms and bonds of a molecule are automatically deleted when their\nparent molecule is destroyed.\n\n<P>\n<dl><dd><pre class=\"verbatim\">#include \"oechem.h\"\n\nusing namespace OEChem;\n\nint main()\n{\n  OEAtomBase *o, *h1, *h2;\n  OEBondBase *b1, *b2;\n  OEMol mol;\n\n  o  = mol.NewAtom(8);\n  h1 = mol.NewAtom(1);\n  h2 = mol.NewAtom(1);\n\n  b1 = mol.NewBond(o,h1,1);\n  b2 = mol.NewBond(o,h2,1);\n  return 0;\n}\n</pre>\n<div class=\"verbatiminput-footer\">\n<a href=\"ch10-1.txt\" type=\"text/plain\">Download as text.</a>\n</div>\n</dd></dl>\n\n<P>\nIn the example source code, the atomic numbers of oxygen, 8, and\nhydrogen, 1, are explicitly encoded in the program.  To make this code\neasier to read and less error prone, OEChem provides symbolic\nconstants for the first 109 elements in the include file <code>elemno.h</code>.\nThis defines the atomic symbols in the C++ namespace OEElemNo with the\nappropriate values.\n\n<P>\n<dl><dd><pre class=\"verbatim\">#include \"oechem.h\"\n\nusing namespace OEChem;\n\nint main()\n{\n  OEAtomBase *o, *h1, *h2;\n  OEBondBase *b1, *b2;\n  OEMol mol;\n\n  o  = mol.NewAtom(OEElemNo::O);\n  h1 = mol.NewAtom(OEElemNo::H);\n  h2 = mol.NewAtom(OEElemNo::H);\n\n  b1 = mol.NewBond(o,h1,1);\n  b2 = mol.NewBond(o,h2,1);\n  return 0;\n}\n</pre>\n<div class=\"verbatiminput-footer\">\n<a href=\"ch10-2.txt\" type=\"text/plain\">Download as text.</a>\n</div>\n</dd></dl>\n\n<P>\n\n<DIV CLASS=\"navigation\">\n<p><hr>\n<table align=\"center\" border=\"0\" width=\"100%\" cellpadding=\"0\" cellspacing=\"0\">\n<tr>\n<td rowspan=2><A HREF=\"node108.html\"><img src=\"../icons/previous.gif\"\n  border=\"0\" alt=\"Previous Page\"></A></td>\n<td rowspan=2><A HREF=\"node108.html\"><img src=\"../icons/up.gif\"\n  border=\"0\" alt=\"Up One Level\"></A></td>\n<td rowspan=2><A HREF=\"node110.html\"><img src=\"../icons/next.gif\"\n  border=\"0\" alt=\"Next Page\"></A></td><td width=\"100%\" rowspan=\"2\"> </td>\n<td valign=\"center\" align=\"right\" height=\"56\" background=\"../icons/title_bkgd.gif\" colspan=4 class=\"navigation\">OEChem - C++ Theory Manual\n<BR>\nVersion 1.3.1</td></tr>\n<tr><td><img src=\"../icons/spacer_1.gif\"\n  border=\"0\" alt=\"\"></td>\n<td><A href=\"contents.html\"><img src=\"../icons/contents.gif\"\n  border=\"0\" alt=\"Contents\"></A></td>\n<td><img src=\"../icons/spacer_2.gif\"\n  border=\"0\" alt=\"\"></td>\n<td><img src=\"../icons/blank_4.gif\"\n  border=\"0\" alt=\"\"></td>\n</tr></table>\n<b class=\"navlabel\">Previous:</b> <a class=\"sectref\" HREF=\"node108.html\">10.
