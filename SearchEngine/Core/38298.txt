There are two different\npossible kinds of requirements, single parameter requirements and multi\nparameter requirements.  Single variable requirements are set using the\n!REQUIRED field in the !PARAMETER record.  A required parameter must either\nbe set by the user or have a default value.  Multi variable requirements\ndepend upon the setting of several variables and are specified with a\ntext configuration file (separate from the text configuration file passed\nto OEConfigure).  The format of the requirement text configuration file\npassed to either <i>OEParseCommandLine</i> or <i>OEParseCommandLineLW</i>\nare described in the API documentation.
