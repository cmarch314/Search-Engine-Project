 \t\t\t\tVerbs and Nouns on the Web    \t\t\t</h1><!-- .entry-title -->\r\n  \t\t\t</header>\r\n\r\n  \t\t\t\r\n  \t\t\t\r\n  \t\t\t<div class=\"entry-content clearfix\">\r\n    \t\t\t<p>John Gehl and Suzanne Douglas write about the verbs and nouns on the Web:</p>\n<blockquote><p>While yesterday&#8217;s Web activities focused on browsing, clicking and<br />\nreading, today&#8217;s Web is all about doing: sharing, socializing,<br />\ncollaborating and, most of all, creating. &#8220;The Web isn&#8217;t so much a place<br />\nanymore,&#8221; says Ross Mayfield, CEO of Socialtext, which helps clients create<br />\ncollaborative sites called wikis. &#8220;They Google, Flickr, blog, contribute to<br />\nWikipedia, Socialtext it, Meetup, post, subscribe, feed, annotate, and<br />\nabove all share. In other words, the Web is increasingly less about places<br />\nand other nouns, but verbs.&#8221; The shift is being driven by the emergence of<br />\nInternet technologies such as Really Simple Syndication (RSS), used for<br />\nblogging, and programming technologies such as Ajax, designed to make Web<br />\napplications almost as fast as traditional desktop programs. In addition,<br />\nnew software called Greasemonkey allows non-techies to add mini-programs<br />\nthat customize the way a particular Web site looks and works on the<br />\nbrowser. For instance, by installing Greasemonkey&#8217;s Book Burro add-on,<br />\nusers perusing books on Amazon will see a price list for that book on other<br />\nsites. &#8220;Applications are no longer software artifacts,&#8221; says Tim O&#8217;Reilly,<br />\nhead of O&#8217;Reilly Media. &#8220;They&#8217;re ongoing services.&#8221; (Business Week 26 Sep 2005)<br />\n<http ://www.businessweek.com/magazine/content/05_39/b3952401.htm></http></p></blockquote>\n<p>All search engines are currently dominated by nouns.  The text processing steps to get to inverted file index ignore everything but &#8216;core&#8217; of nouns.
